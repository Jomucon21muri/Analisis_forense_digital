# Análisis Forense de Sistemas

Objetivo: Analizar sistemas operativos (Windows/Linux/macOS) para identificar compromisos y acciones del atacante.

Descripción: Revisión de artefactos del sistema: cuentas, servicios, tareas programadas, integridad de archivos y registros.

Herramientas comunes: `SIFT`, `Autopsy`, `OSQuery`, `Plaso`, `log2timeline`.

Pasos generales:
- Recolectar imagen del sistema o datos relevantes.
- Revisar cuentas, permisos, servicios, tareas y programas instalados.
- Analizar eventos de seguridad y auditoría.

Artefactos importantes: registros de eventos, cuentas creadas/modificadas, binarios sospechosos, entradas de arranque.
