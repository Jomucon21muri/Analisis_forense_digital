# Análisis Forense de Registros (Logs)

Objetivo: Correlacionar eventos a través de logs para construir una línea temporal y detectar actividad anómala.

Descripción: Recolección y normalización de logs de sistemas, aplicaciones, seguridad y red; búsqueda de patrones y anomalías.

Herramientas comunes: `ELK (Elasticsearch/Logstash/Kibana)`, `Splunk`, `Graylog`, `fluentd`.

Pasos generales:
- Recolectar logs relevantes y asegurar integridad.
- Normalizar y correlacionar eventos por tiempo.
- Buscar indicadores de compromiso e investigar eventos raros.

Artefactos importantes: entradas de eventos, alertas, correlaciones temporales.
