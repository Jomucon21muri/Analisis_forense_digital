# Tipos de Ataques Cibernéticos

Objetivo: Referencia de los principales tipos de ataques, con una breve descripción para uso educativo y de triage.

Tipos comunes:
- Phishing: Engaños por correo o web para obtener credenciales o inducir acciones.
- Malware: Software malicioso (troyanos, puertas traseras, keyloggers).
- Ransomware: Cifrado de datos y demanda de rescate.
- DDoS (Denegación de Servicio Distribuida): Saturación de recursos para dejar servicios inaccesibles.
- Man-in-the-Middle (MitM): Interceptación de comunicaciones entre partes.
- SQL Injection (SQLi): Inyección de comandos en aplicaciones web que consultan bases de datos.
- Cross-Site Scripting (XSS): Inyección de scripts maliciosos en páginas web.
- Credential stuffing / Brute force: Uso de credenciales filtradas o fuerza bruta para acceder a cuentas.
- Supply Chain Attacks: Compromiso de un proveedor o dependencia para atacar a terceros.
- Insider Threat: Amenazas originadas desde dentro de la organización.
- Zero-day Exploits: Aprovechamiento de vulnerabilidades no corregidas públicamente.
