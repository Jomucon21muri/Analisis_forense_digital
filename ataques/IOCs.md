# IOCs de ejemplo - Catálogo de Ataques

## Phishing
- Dominios: phishing-example[.]com
- URLs: http://phish.example/login

## Malware / Ransomware
- Hashes (ejemplo): SHA256: 3a7bd3e2360a3d9f5d7c9d8c1f1a2b3c4d5e6f7a8b9c0d1e2f3a4b5c6d7e8f9

## DDoS
- IPs de origen (ejemplo): 198.51.100.100, 198.51.100.101

## SQLi / XSS
- URLs de prueba: /vulnerable.php?id=1' OR '1'='1

## Notas
- Usar estos ejemplos para crear reglas de detección y ejercicios de triage.
